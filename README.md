# Font-awesome Library



这是一个纯净的Font-awesome (最新6.7.2)免费图标项目，快速查找Font-awesome图标，支持模糊搜索。

This is a clean Font Awesome free icons project for quickly searching Font Awesome icons, with support for fuzzy search.
